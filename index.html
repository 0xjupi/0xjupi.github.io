iGebeya Buy & Sell – Details

Overview

iGebeya is a local-first, low-bandwidth-friendly Telegram Mini App designed to simplify digital classifieds in Ethiopia. Users can list items for sale, add images, and categorize them with tags. Listings can be promoted using boost tokens or rewarded ad views.

It includes secure identity verification (powered by 101-Verify which is my undergoing KYC project), a reseller system with payout logic, and Telegram-native interactions for seamless in-app communication. The backend is built with FastAPI and MySQL, and the UI is optimized for Telegram WebView UX.

The mini app gathers users interaction data through time and uses it to recommend related items. The platform is 100% free where users can earn free boosts by turning ads on, checking in daily, and completing other tasks. These boosts can be used to boost listed items.

When users want to buy more boosts they can simply pay using telegram integrated crypto wallets. The crypto transaction is handled in a secure encrypted way on "TON Blockchain" and doesn't expose any users' data.

Admins can monitor users' interactions, items, profile verification approval and etc on another telegram mini app dedicated only for admins.

Research Papers

3D U-Net-Based Brain Tumor Semantic Segmentation Using a Modified Data Generator

Published in IMA: International Journal of Imaging Systems and Technology, 2025

This research introduces a novel 3D U-Net architecture enhanced with a custom data generator for segmenting gliomas from MRI scans using the BraTS 2020 dataset.

The data generator performs real-time augmentation to improve generalization and reduce overfitting. Our model achieved dice scores of 82.2% (tumor core), 90.3% (whole tumor), and 77.8% (enhancing tumor), demonstrating strong robustness across tumor subtypes.

The approach improves segmentation reliability and supports more accurate treatment planning for glioblastoma patients.
